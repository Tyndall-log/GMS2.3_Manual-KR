물리 _ 입자 _ 그룹 _ 시작

이 기능을 사용하면 방에 파티클 그룹을 만들 수 있습니다. 파티클 그룹은 특정 폼을 생성하는 데 필요한 파티클이 그룹으로 한 번에 생성되어 사용되는 플래그로 설정된 다양한 속성을 가진 시뮬레이션 된 소프트 바디를 생성 할 수있는 파티클입니다. "플래그"는 입자 상수 조합의 반환 값입니다 (자세한 내용은 physics_particle_create () 함수 참조). "groupFlags"는 다음 그룹 상수 의 조합입니다 (또는 플래그가없는 경우 0).

 

일정한 기술
phy_particle_group_flag_solid 고체 입자 그룹은 다른 픽스쳐가 그 안에 들어 가지 못하게합니다. 어떤 것이라도 침투하면, 고체 입자 그룹은 문제가되는 고정구를 다시 표면으로 밀어 내고, 고체 입자 그룹은 특히 강한 반발력을 갖습니다.
phy_particle_group_flag_rigid 강성 입자 그룹은 다른 고정 장치와 충돌하더라도 모양이 변경되지 않는 그룹입니다.

 

이 그룹 플래그는 비트 마스킹을 사용하여 최종 출력 값을 생성 한 다음 파티클 플래그에 의해 설정된 것과는 별도로 그룹의 다른 기본 속성을 설정하기 위해 검사됩니다. 비트 단위 또는 "|"를 사용하여 이러한 상수 중 하나 또는 둘 다를 사용할 수 없습니다 적절한 비트를 마스킹합니다 (아래 예 참조).

플래그 이외에도, 파티클 그룹을 생성하기 위해 실내의 위치를 ​​(주어진 위치는 완성 된 그룹의 질량 중심이 될 것입니다), 초기 수평 및 수직 속도, 각도 속도를 설정할 수 있습니다. 이동하는 회전 그룹을 만들 수 있습니다.

physics_particle_draw () 함수를 사용하여 파티클을 그릴 때 사용되는 값으로 그룹 내 모든 파티클의 초기 색상과 알파를 설정할 수도 있습니다. 그런 다음 그룹에서 입자의 응집력을 설정합니다. 이는 입자가 서로 "강하게 붙어"있는 또 다른 방법입니다. 강도는 0과 1 사이의 값일 수 있습니다. 여기서 0은 응집력이없고 1은 최대 응집력입니다.

마지막으로 파티클 그룹의 카테고리를 제공해야합니다. 이 값은 비슷한 속성을 가지려는 각 입자 그룹에 제공하는 임의의 정수 값이며, 나중에 특정 범주 및 기타 여러 가지에 대해 플래그를 설정하는 데 사용할 수 있습니다.

참고 : 시뮬레이션에 예약되어 있고 다른 기능 (예 : 그림)에서 모든 범주를 선택하는 데 사용되는 0 을 제외하고 범주에 정수 값을 사용할 수 있습니다.

이 함수를 호출한다고해서 물리 실에 그룹이 생성되지는 않습니다 . 이 기능은 단순히 프로세스를 시작하고 그룹의 기본 입자 속성을 설정합니다. 파티클 그룹을 실제로 만들려면 폼 그룹을 만들고 (아래 예 참조) physics_particle_group_end () 함수를 호출하면 생성됩니다.

참고 : 설정 한 파티클 유형 플래그는 파티클 그룹의 응집력에도 영향을 미칩니다. 예를 들어 파티클이 워터 타입 인 것으로 플래그를 지정하면 그룹이 다른 조명기와 충돌 할 때 분리되는 동안 입자를 스프링 유형으로 표시하면 충돌시 응집력이 유지됩니다.

 

통사론:

physics_particle_group_begin(flags, groupflags, x, y, ang, xv, yv, ang_velocity, col, alpha, strength, category)

논의 기술
깃발 파티클에 설정할 플래그입니다.
그룹 플래그 파티클에 설정할 그룹 플래그입니다 (아래 설명 참조).
엑스 파티클 그룹을 생성 할 x 위치입니다.
와이 파티클 그룹을 생성 할 y 위치입니다.
중앙 그룹을 만들 때 회전 할 각도 (도)입니다.
xv 초기 수평 속도.
yv 초기 수직 속도.
ang_velocity 초기 각속도.
안부 입자 그룹에 사용할 기본 색상입니다.
알파 입자 그룹에 사용할 기본 알파입니다.
힘 그룹 내 입자 사이의 응집력을 설정하십시오.
범주 그룹이 속한 사용자 정의 범주.

 

보고:

N/A

 

예:

var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile;
var groupflags = phy_particle_group_flag_solid | phy_particle_group_flag_rigid;
physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, c_white, 1, 1, 2);
physics_particle_group_circle(100);
mLastGroup = physics_particle_group_end();

위의 코드는 입자 유형 및 입자 그룹 속성에 대한 플래그를 변수에 저장 한 다음이를 사용하여 마우스 위치에서 반지름이 100px 인 원형 입자 그룹을 만듭니다.
