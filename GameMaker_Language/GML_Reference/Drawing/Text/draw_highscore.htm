draw_highscore

이 함수는 현제 내부에 저장된 highscore 리스트를 현제의 글꼴, 알파값(투명값)과 색상을 지정된 사각형내에 그릴겁니다. 여기서 문자열이 그려질 사각형공간의 좌측상단의 좌표와 좌측하단의 좌표를 넣어주면 됩니다, 이러면 게임메이커 스튜디오2는 자동적으로 간격과 위치를 잡아줍니다.

 

문법:

draw_highscore( x1, y1, x2, y2 );

Argument Description
x1 사각형 좌측의 x 좌표값.
y1 사각형 상단의 y 좌표값.
x2 사각형 우측의 x 좌표값.
y2 사각형 하단의 y 좌표값.

 

반환값:

N/A

 

예시:

draw_highscore(100, 100, room_width - 100, room_height - 100);

이것은 100픽셀의 경계를 가지는 사각형내에 highscore의 표를 그릴것 입니다.
