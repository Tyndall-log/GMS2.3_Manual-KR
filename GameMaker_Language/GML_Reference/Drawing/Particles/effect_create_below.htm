effect_create_below

이 기능을 사용하면 방의 모든 인스턴스 아래에 간단한 효과를 만들 수 있습니다 (실제로 깊이 100000에서 생성됨). 효과가 다른 ef_rain 또는 ef_snow 인 경우 x / y 위치를 정의하여 효과를 만들 수 있으며 크기는 0, 1 또는 2의 값일 수 있습니다. 여기서 0은 작고 1은 중간이고 2는 큰 .

파티클 시스템 인덱스에 적절한 값을 갖는 part_system_automatic_draw () 및 part_system_automatic_update () 함수를 사용하여 이러한 효과를 사용하여 그리기를 켜거나 끌 수 있으며 그리기를 일시 중지 할 수 있습니다 (여기서 0은 효과에 대한 것임) 아래 및 1은 위의 효과를 나타냅니다.

다른 입자 종류에 사용 가능한 상수는 다음과 같습니다.

일정한 예 기술
ef_cloud 다양한 크기의 임의의 구름 입자를 생성하는 효과
ef_ellipse 확장 타원을 만드는 효과
ef_explosion 페이딩 폭발을 확장시키는 효과
ef_firework 불꽃 폭발을 일으키기 위해 여러 개의 작은 입자를 만드는 효과
ef_flare 눈부신 빛을 발하는 화려한 점을 만드는 효과
ef_rain 화면 상단에서 비가 내리는 효과
ef_ring 확장 및 페이딩 서클을 생성하는 효과
ef_smoke 연기가 거의 나지 않는 효과
ef_smokeup 연기 연기를 생성하여 화면을 상승시키는 효과
ef_snow 화면 아래로 떨어지는 여러 개의 눈 입자를 생성하는 효과
ef_spark 작은 불꽃을 발생시키는 효과
ef_star 별 입자를 생성하는 효과

 

통사론:

effect_create_below(kind, x, y, size, colour);

논의 기술
종류 효과의 종류 (위에 나열된 상수 중 하나를 사용하십시오).
엑스 해당되는 경우 효과의 x 위치.
와이 해당되는 경우 효과의 y 위치.
크기 효과의 크기입니다.
색깔 효과의 색상입니다.

예:

if speed > 0
    {
    effect_create_below(ef_smoke, x, y, 0, c_gray);
    }

위의 코드는 인스턴스 x, y 좌표에서 인스턴스 속도가 0보다 큰 단계마다 작은 회색 연기 퍼프를 만듭니다.
