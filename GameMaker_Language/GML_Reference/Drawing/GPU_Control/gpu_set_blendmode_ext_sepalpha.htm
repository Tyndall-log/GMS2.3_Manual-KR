gpu_set_blendmode_ext_sepalpha

GameMaker Studio 2가 픽셀을 그리면 소스 색상 (그린 픽셀의 색상)과 대상 색상 (이미 그리는 픽셀에있는 색상)이 있으므로 픽셀의 새로운 색상, 소스 및 대상 색상은 선택한 블렌드 모드에 따라 계산됩니다. 색상의 각 구성 요소는 0과 1 사이의 부동 소수점 값으로 저장되며, 새로운 색상은 소스 색상의 각 구성 요소에 몇 가지 요소를 곱하고 대상 색상의 각 구성 요소에 다른 요소를 곱한 다음 구성 요소별로 결과를 함께 제공합니다.

이 기능을 사용하면 RGB 구성 요소와 알파 구성 요소에 대해 서로 다른 혼합 모드 요소의 다른 부분을 선택하여 사용자 정의 혼합 모드를 만들기 위해 함께 고려해야하는 서로 다른 구성 요소 부분을 설정할 수 있습니다. 따라서 "src"및 "dest"인수는 처음 3 개의 혼합 모드 값 (RGB)을 사용하고 "alphasrc"및 "alphadest"인수는 혼합 모드 요소의 마지막 부분을 차지합니다. 예를 들어, RGB 소스가 bm_zero 이면 (0, 0, 0)이 사용됩니다. 알파 소스를 bm_one으로 설정하면 (1)이 사용되어 최종 소스 블렌드 모드는 (0, 0, 0, 1)이됩니다.

다음 차트는 각 혼합 모드의 RGBA 값을 보여줍니다. 여기서 (Rs, Gs, Bs, As)는 소스 값으로 간주되고 (Rd, Gd, Bd, Ad)는 대상 값입니다. 사용 가능한 11 개의 상수는이 함수에서 사용될 때 소스 또는 대상 (또는 둘 다) 일 수 있습니다.

일정한 블렌드 팩터 (빨강, 녹색, 파랑, 알파)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-R, 1-G, 1-B, 1-A)
bm_src_alpha (As, As, As, As)
bm_inv_src_alpha (1-As, 1-As, 1-As, 1-As)
bm_dest_alpha (광고, 광고, 광고, 광고)
bm_inv_dest_alpha (1- 광고, 1- 광고, 1- 광고, 1- 광고)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1) 여기서 f = min (As, 1-Ad)

 

이 함수에 4 개의 개별 인수를 제공하거나 인수 배열을 제공 할 수 있습니다 ( 예 : gpu_get_blendmode_ext_sepalpha () 함수에 의해 리턴 됨). 배열을 제공하는 경우 다음 네 가지 요소가 있어야합니다.

  • [0] = 소스 블렌드 모드에서 알파 성분을 뺀 값 (기본값은 bm_src_alpha (As, As, As))
  • [1] = 대상 블렌드 모드에서 알파 성분을 뺀 값 (기본값은 bm_inv_src_alpha (1-As, 1-As, 1-As))
  • [2] = 소스 알파 구성 요소 (기본값은 bm_src_alpha (As))
  • [3] = 대상 알파 구성 요소 (기본값은 bm_inv_src_alpha (1-As))

중대한! WebGL을 사용하지 않는 HTML5는 다음 모드를 올바르게 표시 할 수 없습니다 .

  • bm_src_colour
  • bm_inv_src_colour
  • bm_dest_colour
  • bm_inv_dest_colour
  • bm_src_alpha_sat

 

 

통사론:

gpu_set_blendmode(src, dest, alphasrc, alphadest);

논의 기술
src RGB의 소스 블렌드 모드 계수 (위 상수 참조).
대상 RGB의 대상 혼합 모드 계수 (위 상수 참조)
alphasrc 소스 블렌드 모드 팩터 알파 (위 상수 참조).
알파 대상 블렌드 모드 팩터 알파 (위 상수 참조)

 

보고:

N/A

 

예:

var bm = gpu_get_blendmode_ext_sepalpha();
bm[0] = bm_src_alpha;
bm[1] = bm_inv_src_alpha;
gpu_set_blendmode_ext_sepalpha(bm);

위의 코드는 로컬 배열을 만들고 별도의 알파 값을 가진 현재 확장 혼합 모드를 가져옵니다. 그런 다음이 배열은 조작되어 블렌드 모드를 설정하도록 반환됩니다.
