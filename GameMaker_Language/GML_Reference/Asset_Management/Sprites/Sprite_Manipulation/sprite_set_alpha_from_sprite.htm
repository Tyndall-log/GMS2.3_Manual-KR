sprite_set_alpha_from_sprite

이 함수는 하나의 스프라이트의 값 / 채도를 사용하여 대상 스프라이트의 알파와 곱합니다. 이상적으로 새 알파 맵을 생성하는 데 사용되는 스프라이트는 그레이 스케일이어야합니다. 흰색 영역의 알파 값은 1 (불투명), 검은 색 영역은 알파 0 (투명), 회색 영역은 알파입니다 알파를 설정하는 스프라이트 는 영구 리소스 일 수 없으며 알파를 설정하는 데 사용하는 스프라이트는 오류가 발생할 수 있습니다. 즉, 함수가 제대로 작동하려면 어떤 방식 으로든 "즉석에서"완전히 새로운 두 개의 스프라이트를 만들어야합니다. 표면을 사용하거나 화면에서 스프라이트를 만들어 달성 할 수 있습니다 (아래 코드 예제는 표면으로 효과를 얻는 방법을 보여줍니다).

참고 :이 함수는 비트 맵 스프라이트에만 유용하며 SWF 또는 JSON (스파인) 스프라이트에서는 작동하지 않습니다.

다음은이 기능의 작동 방식을 보여주는 이미지입니다.

 

구문:

sprite_set_alpha_from_sprite(ind, spr);

논의 기술
산업 투명도 맵을 변경할 스프라이트의 인덱스입니다.
spr 투명도 맵 데이터를 가져올 스프라이트의 인덱스입니다.

확장 된 예 :

이 확장 예제에서는 두 개의 스프라이트 리소스에서 특수 효과 스프라이트를 만드는 방법을 보여줍니다. 시작하려면 인스턴스의 이벤트 생성에 컨트롤러 변수가 있어야합니다. 이 변수는 스프라이트가 한 번만 생성되도록하기위한 것입니다.

spr_create = true;

완료되면 Draw Event 로 이동하여 다음 코드를 추가해야합니다.

if spr_create
    {
    spr_create = false;
    var _sw = sprite_get_width(spr_Explosion_Alpha);
    var _sh = sprite_get_height(spr_Explosion_Alpha);
    var _xo = sprite_get_xoffset(spr_Gradient);
    var _yo = sprite_get_yoffset(spr_Gradient);
    var _surf = surface_create(_sw, _sh);
    surface_set_target(_surf);
    draw_clear_alpha(c_black, 0);
    draw_sprite(spr_Gradient, 0, 0, 0);
    sprite_index = sprite_create_from_surface(_surf, 0, 0, _sw, _sh, false, false, _xo, _yo);
    draw_clear_alpha(c_black, 0);
    draw_sprite(spr_Explosion_Alpha, 0, _xo, _yo);
    var _spr = sprite_create_from_surface(_surf, 0, 0, _sw, _sh, false, false, _xo, _yo);
    surface_reset_target();
    surface_free(_surf);
    sprite_set_alpha_from_sprite(sprite_index, _spr);
    sprite_delete(_spr);
    }
draw_sprite(sprite_index, 0, x, y);

위의 코드는 먼저 제어 변수가 true인지 여부를 확인하고, 그렇지 않은 경우 코드를 실행하여 스프라이트 알파를 설정하고 변수가 false로 설정되어 코드가 두 번 이상 실행되지 않도록합니다. 이 경우 사용하려는 알파 마스크 크기의 표면을 만든 다음 알파를 표면으로 변경해야 할 스프라이트를 그린 후 표면에서 새 스프라이트를 만들어 객체의 sprite_index 로 지정합니다. . 그런 다음 코드는 표면을 지우고 알파 맵 스프라이트를 그려서 다른 새 스프라이트를 만들어 인덱스를 로컬 변수에 저장합니다. 마지막으로 객체에 할당 된 새 스프라이트는이 임시 스프라이트에서 알파 세트를 가져오고 표면이 해제되고 임시 스프라이트가 제거됩니다.

여기서해야 할 마지막 일이 있는데, 사용하지 않을 때 오브젝트에서 만들어진 스프라이트를 메모리에서 제거하는 것입니다. 이는 다음과 같이 Destroy Event 또는 Room End Event 에서 수행됩니다.

sprite_delete(sprite_index);
