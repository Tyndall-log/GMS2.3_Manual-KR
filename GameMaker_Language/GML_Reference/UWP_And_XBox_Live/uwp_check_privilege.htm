uwp_check_privilege

이 기능을 사용하면 지정된 사용자에게 선택된 권한이 있는지 확인할 수 있습니다. 이 기능은 권한이 활성화되어 있는지 여부에 따라 시스템 비동기 이벤트에서 true 또는 false를 반환하며, try_resolution 인수를 true로 설정하고 권한이 활성화되어 있지 않으면 시스템 대화 상자가 열립니다 (게임 일시 중지) )를 사용하여 계정 또는 권한을 얻는 데 필요한 모든 것을 업그레이드하라는 메시지를 표시합니다. 사용자가 필요한 옵션을 활성화하면이 함수는 true를 반환합니다.

이 기능을 사용하려면 권한 요청에 다음 상수 중 하나를 사용해야합니다.

 

일정한 기술
uwp_privilege_internet_browsing 인터넷 브라우징이 허용되는지 확인
uwp_privilege_social_network_sharing 소셜 네트워크 공유가 허용되는지 확인
uwp_privilege_share_kinect_content Kinect 컨트롤러 공유가 허용되는지 확인
uwp_privilege_video_communications 비디오 통신이 허용되는지 확인
uwp_privilege_communications 인터넷 통신이 허용되는지 확인
uwp_privilege_user_created_content 사용자 제작 콘텐츠에 대한 액세스가 허용되는지 확인
uwp_privilege_multiplayer_sessions 온라인 멀티 플레이어 세션이 허용되는지 확인
uwp_privilege_sessions 온라인 세션이 허용되는지 확인
uwp_privilege_fitness_upload 피트니스 데이터 업로드가 허용되는지 확인

 

함수가 호출되면 XBox One에서 콜백 결과가 수신 될 때 비동기 시스템 이벤트를 트리거하며 특수 DS 맵 async_load가 있습니다. 그런 다음이 키를 다음 키에 대해 구문 분석해야합니다.

  • " event_type "-이벤트가이 함수에 의해 트리거 된 경우 "check_privilege_result"값을 보유해야합니다.

이벤트 유형이이 기능과 관련이있는 경우 XBox에서 함수가 호출 될 때 다음과 같은 추가 키가 있습니다.

  • " result "-다음 상수 중 하나 이상이됩니다 (결합 된 값). 일정한 기술 값 uwp_privilege_result_no_issue 사용자와 관련된 권한 문제가 없습니다 0 uwp_privilege_result_purchase_required 사용자는 액세스를 위해 일반적으로 추가 요금을 구매해야합니다. 1 uwp_privilege_result_aborted 수표가 중단되었습니다 2 uwp_privilege_result_banned 사용자가 금지되었습니다 4 uwp_privilege_result_restricted 일반적으로 자녀 보호 기능을 통해 사용자의 액세스가 제한됩니다. 8

  • " privilege "-함수에 사용하기 위해 위에 나열된 상수 중 하나이므로 다중 검사의 경우 요청한 권한을 확인할 수 있습니다.

프로젝트가 XBox One이 아닌 UWP 플랫폼에서 실행되는 경우 비동기 이벤트가 트리거 되지 않고 대신 함수가 -1, 0 또는 1로 즉시 리턴됩니다.

  • -1은 -1은 사용자에게 권한이 없음을 의미합니다.
  • 0은 함수가 사용자를 쿼리 할 수 ​​없음을 의미합니다
  • 1은 사용자에게 권한이 있음을 의미합니다.

 

구문:

uwp_check_privilege(user_id, privilege, attempt_resolution);

논의 기술
user_id 확인할 사용자 ID 포인터.
특권 확인할 특권 상수 (위의 설명 참조).
시도 _ 해결 true로 설정된 경우 해상도 시도를 활성화하거나 false로 설정된 경우 무시하십시오.

 

보고:

Integer

 

예:

var user_one = xboxlive_get_user(0);
if (xboxlive_user_is_signed_in(user_one))
    {
    uwp_check_privilege(user_one, xboxlive_privilege_multiplayer_sessions, true);
    }

위의 내용은 사용자가 로그인했는지 확인하고 사용자인지 확인하여 멀티 플레이어 세션 권한을 확인합니다.
