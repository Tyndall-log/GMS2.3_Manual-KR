url_open_full

이 함수는 유저가 선택한 브라우저로 해당 URL을 열어줍니다. 또는 HTML5 모듈을 사용하고 있는경우, 현재 작동하고 있는 브라우저에서 URL이 열립니다.

"target" 파라미터는 Javascript에서 open() 함수의 "name" 파라미터와 동일한 역할을 합니다.
"options" 파라미터는 Javascript에서 open() 함수의 "specs" 파라미터와 동일한 역할을 합니다.

※주의: target에서 "_self" 값은 브라우저에서 자동으로 차단하거나, 유저가 이를 허용할지 묻는 안내가 뜰 수 있습니다.
※주의: options 값은 모든 브라우저에서 적용되지 않을 수 있습니다.

가능한 target 종류:

 

Target 설명
_blank 링크된 문서를 새 창 혹은 새 탭에서 엽니다. (이 작업은 팝업에 경우 유저에게 막힐 수 있습니다. 대신 clickable_* 함수를 사용할 수 있습니다.)
_self 링크된 문서를 함수가 실행된 창에서 엽니다. (기본)
_parent 링크된 문서를 부모 창에서 엽니다.
_top 링크된 문서를 최상위 윈도우에 표시합니다.

 

가능한 options 종류:

 

파라미터 설명
'height=[px]' 윈도우의 높이 (최소 100 이상)
'width=[px]' 윈도우의 너비 (최소 100 이상)
'left=[px]' 윈도우의 x축 위치를 지정
'top=[px]' 윈도우의 y축 위치를 지정 (IE only)
'location=[boolean]' URL 입력란을 표시할지 여부입니다 (default is 1).
'menubar=[boolean]' 메뉴바를 표시할지 여부입니다 (default is 1).
'resizable=[boolean]' 창의 크기를 조절할 수 있는 여부를 결정 (default is 1).
'scrollbars=[boolean]' 스크롤 바를 표시할지 여부입니다 (default is 1).
'status=[boolean]' 상태 표시줄을 표시할지 여부입니다 (default is 1).
'titlebar=[boolean]' 제목 표시줄을 표시할지 여부입니다. 호출하는 응용 프로그램이 HTML 응용 프로그램이거나 신뢰할 수 있는 대화 상자가 아니면 무시됩니다. (default is 1)
'toolbar=[boolean]' 도구상자(toolbar)을 표시할지 여부입니다 (default is yes).

 

문법:

url_open_full(url, target, options);

인자 설명
url 이동하고자 하는 URL (웹 사이트 주소)
target URL을 열고자 하는 위치
options 브라우저 옵션 (설명 참조)

 

반환값:

없음

 

예시:

url_open_full("http://yoyogames.com", "_blank", "resizable=0, height=200, scrollbars=0");

이 예시는 요요게임즈 홈페이지를, 사이즈 조절이 불가하고 높이가 200px이며 스크롤바가 없는 새 창에서 열어줍니다.
