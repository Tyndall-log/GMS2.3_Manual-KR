url_open

이 예시는 유저가 선택한 브라우저로 해당 URL을 열어줍니다. 또는 HTML5 모듈을 사용하고 있는경우, 현재 작동하고 있는 브라우저에서 URL이 열립니다.

 

문법:

url_open(url);

인자 설명
url 이동하고자 하는 URL (웹 사이트 주소)

 

반환값:

없음

 

예시:

url_open("http://yoyogames.com");

이 함수는 현재 윈도우에서 요요게임즈 홈페이지를 열 것입니다.
