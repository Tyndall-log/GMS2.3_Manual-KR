url_open_ext

이 함수는 유저가 선택한 브라우저로 해당 URL을 열어줍니다. 또는 HTML5 모듈을 사용하고 있는경우, 현재 작동하고 있는 브라우저에서 URL이 열립니다. "target" 파라메터는 Javascript에서 open() 함수의 "name" 파라메터와 동일한 역할을 합니다.

※주의: "_self" 값은 브라우저에서 자동으로 차단하거나, 유저가 이를 허용할지 묻는 안내가 뜰 수 있습니다.

가능한 target 종류:

 

Target 설명
_blank 링크된 문서를 새 창 혹은 새 탭에서 엽니다. (이 작업은 팝업에 경우 유저에게 막힐 수 있습니다. 대신 clickable_* 함수를 사용할 수 있습니다.)
_self 링크된 문서를 함수가 실행된 창에서 엽니다. (기본)
_parent 링크된 문서를 부모 창에서 엽니다.
_top 링크된 문서를 최상위 윈도우에 표시합니다.

 

문법:

url_open_ext(url, target);

인자 설명
url 이동하고자 하는 URL (웹 사이트 주소)
target URL을 열고자 하는 위치

 

반환값:

없음

 

예시:

url_open_ext("http://yoyogames.com", "_blank");

이 예시는 새 페이지에서 요요게임즈 홈페이지를 열어 줍니다.
