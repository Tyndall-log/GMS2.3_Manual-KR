clickable_add

이 기능을 사용하면 게임 캔버스 외부에서 브라우저 창 자체 내에서 "버튼"으로 작동하는 클릭 가능한 사용자 정의 아이콘을 만들 수 있습니다. 이 함수는 새 아이콘을 참조하는 모든 추가 버튼 기능에 사용되도록 변수에 저장해야하는 id 값을 반환합니다. 이 기능은 GameMaker Studio 2 자체를 통해 버튼을 클릭 할 때 브라우저가 팝업 창을 만들지 못하게하는 데 매우 유용합니다.

먼저 브라우저 창의 (0,0)에 상대적인 위치 (왼쪽 위 모서리)를 설정 한 다음 버튼 아이콘에 원하는 스프라이트에 대한 텍스처 페이지 항목을 제공해야합니다 ( sprite_get_tpe () 참조) . 그런 다음 URL을 열거 나 GameMaker Studio 2의 Asset Browser에서 스크립트를 열거 나 실행할 수 있습니다. 지정하는 "target"매개 변수는 open () 메소드를 사용할 때 표준 JavaScript "name"값과 동일합니다 ( '_self'를 제외한 모든 것이 브라우저를 차단하거나 원하는 경우 사용자에게 요청할 수 있음에 유의하십시오) "params"인수는 새 창 / 탭에 표시 할 속성을 제어하기위한 JavaScript "specs"매개 변수와 동일합니다 (모든 브라우저가 모든 기능을 지원하지는 않습니다). 아이콘을 사용하여 내부 스크립트를 실행하는 경우이 마지막 두 인수는 빈 문자열 ""로 남겨 둘 수 있습니다.

유효한 목표는 다음과 같습니다.

 

표적 기술
_공백 링크 된 문서를 새 창이나 탭에서 엽니 다.
_본인 클릭 한 것과 동일한 프레임에서 링크 된 문서를 엽니 다 (기본값).
_부모의 연결된 문서를 부모 프레임에서 엽니 다.
_상단 링크 된 문서를 창의 전체 본문에서 엽니 다.

 

유효한 매개 변수는 다음과 같습니다.

 

매개 변수 기술
'height = [ px ]' 최소값이 100 인 창의 높이입니다.
'width = [ px ]' 창의 너비로 최소값은 100입니다.
'left = [ px ]' 창의 왼쪽 위치
'top = [ px ]' 창의 상단 위치 (IE 만 해당)
'location = [ 부울 ]' 주소 필드 표시 여부 (기본값은 1)
'menubar = [ 부울 ]' 메뉴 표시 줄 표시 여부 (기본값은 1)
'resizable = [ 부울 ]' 창의 크기를 조정할 수 있는지 여부 (기본값은 1)
'scrollbars = [ 부울 ]' 스크롤 막대를 표시할지 여부 (기본값은 1)
'status = [ 부울 ]' 상태 표시 줄을 추가할지 여부 (기본값은 1)
'titlebar = [ 부울 ]' 제목 표시 줄을 표시할지 여부입니다. 호출하는 응용 프로그램이 HTML 응용 프로그램이거나 신뢰할 수있는 대화 상자가 아닌 경우 (기본값은 1) 무시됩니다.
'toolbar = [ 부울 ]' 브라우저 툴바를 표시할지 여부 (기본값은 yes)

전달한 URL 인수가 스크립트 함수 의 이름 인 경우 (예 : "http : //"로 시작하지 않음) 스크립트가 gmcallback_ * 이면 직접 피드를 가져올 수 있습니다. html5 이벤트 GameMaker Studio 2 스크립트 함수 이름은 gmcallback_로 시작하면 난독 화 되지 않습니다 .

 

구문:

clickable_add(x, y, tpe, url, target, params);

논의 기술
엑스 윈도우 내의 x 위치.
와이 윈도우 내의 y 위치.
tpe 사용할 스프라이트의 텍스처 페이지 항목입니다.
url 링크 할 URL (웹 사이트 주소) 또는 실행할 스크립트입니다.
표적 URL을 열기위한 대상 영역입니다 (설명 참조).
매개 변수 아이콘의 다양한 매개 변수 (설명 참조).

 

보고:

Real

 

예:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

위 코드는 게임 캔버스가 실행중인 페이지의 위치 (32, 32)에 클릭 가능한 DOM 아이콘을 만듭니다. 아이콘은 텍스처 페이지에서 "spr_MS_Home"으로 참조 된 스프라이트를 사용하며, 클릭하면 아이콘은 지정된 URL에 대해 새 창을 열고 해당 창에 대해 정의 된 속성을 갖습니다.
