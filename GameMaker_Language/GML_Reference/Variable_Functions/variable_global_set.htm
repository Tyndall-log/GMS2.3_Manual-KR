variable_global_set

이 함수를 사용하시면 인자로 넣어준 전역변수의 값을 설정할 수 있습니다. 함수의 인자값은 설정하길 원하는 전역변수의 이름(문자열 데이터)과 설정하고 싶은 값(data type항목에 부합되면 어떤 형태든 상관없음)을 넣어주시면 됩니다. 만약 인자값으로 넣어준 전역변수의 이름이 게임내에 존재하지 않다면 해당 변수이름의 값을 생성하고 값을 부여하게 됩니다.

 

문법:

variable_global_set(name, val);

인자값(Argument) 설명(Description)
name 값을 설정하고 싶은 전역변수의 이름(문자열 데이터)
val 전역변수를 설정하고 싶은 값

 

반환값:

N/A

 

예제:

if !variable_global_exists("enemy_num")
   {
   variable_global_set("enemy_num", 0);
   }

위 코드는 전역변수가 존재하는지 여부를 판단하고, 존재하지 않는다면 해당 변수이름으로 생성하고 0으로 값을 부여해줍니다.
