variable_global_exists

이 함수를 사용하면 전역 영역 변수(global scope variable)가 존재하는지 여부를 확인하실 수 있습니다. 확인하고 싶은 전역변수의 이름(문자열)을 전역변수의 인자값으로 주게 되고(아래 코드 예제 참조), 함수는 인자값으로 넣어준 전역변수가 존재한다면 true를 반환하게 되고, 그렇지않다면 false를 반환하게 됩니다.

 

문법:

variable_global_exists(name);

인자값(Argument) 설명(Description)
name 확인하고 싶은 전역변수의 이름(문자열)

 

반환값:

Boolean

 

예제:

if !variable_global_exists("enemy_num"))
{
global.enemy_num = instance_number(obj_Enemey_Parent);
}

위 코드는 "enemy_num"으로 선언된 전역변수가 존재하는지를 확인하고, 그렇지 않다면 새로운 enemy_num이라는 변수를 만들어서 값을 지정해주고 있습니다.
