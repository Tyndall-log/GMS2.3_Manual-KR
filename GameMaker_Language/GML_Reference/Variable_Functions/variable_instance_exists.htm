variable_instance_exists

이 함수를 사용하면 개체 영역 변수(Instance scope variable)가 존재하는지 여부를 확인할 수 있습니다. 문자열로써 체크되는 변수이름뿐만 아니라 고유한 개체 ID(Instance ID)값(룸 에디터의 Instance Properties 에서 찾을 수 있거나, 함수 instance_create_layer() 를 호출할 때 반환되는 값)을 인자값으로 넣어줘야 합니다.(샘플 코드는 아래에서 확인할 수 있다.) 이 함수는 해당 인스턴스에 주어진 이름의 변수가 존재하면 ture를 반환을 하고, 그렇지 않다면 false를 반환하게 된다.

 

문법:

variable_instance_exists(instance_id, name);

인자 값(Argument) 서술(Description)
instance_id 확인할 인스턴스의 고유한 ID 값
name 확인할 변수의 이름(문자형 데이터)

 

반환값:

Boolean

 

예제:

if !variable_instance_exists(id, "shields"))
{
shields = 0;
}

위 코드에서 변수 "shields"가 해당 인스턴스에 존재하는지 확인하고, 존재하지 않다면 0으로 초기화해서 만드는걸 확인할 수 있습니다.
