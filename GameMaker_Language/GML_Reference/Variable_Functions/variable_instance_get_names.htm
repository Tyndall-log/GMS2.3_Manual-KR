variable_instance_get_names

이 함수를 사용하시면 특정 개체의 변수이름, 또는 게임 내에 전역(global)변수가 들어있는 배열을 검색할 수 있습니다. 함수 인자에 개체 ID값을 넣게 되면, 배열의 각 항목들은 개체에서 만들어진 instance scope 변수에 대응되는 변수이름의 문자열이 됩니다. 하지만 함수 인자값에 "global"이란 키워드를 넣게되면, 배열의 각 항목들은 변수에 global scope 대응되는 변수이름의 문자열이 됩니다.

 

문법:

variable_instance_get_names(instance_id/global);

인자값(Argument) 서술(Description)
instance_id/global 확인 할 특정 개체의 ID값 또는 "global"키워드

 

반환값:

Array (each entry is a string)

 

예제:

var str = "";
var array = variable_instance_get_names(id);
show_debug_message("Variables for " + object_get_name(object_index) + string(id));
for (var i = 0; i < array_length(array); i++;)
    {
    str = array[i] + ":" + string(variable_instance_get(id, array[i]));
    show_debug_message(str);
    }

위 코드는 코드블럭에서 동작하는 모든 개체 범위 변수의 배열을 검색하고, 해당 값들을 디버그 출력으로 보여줍니다.
