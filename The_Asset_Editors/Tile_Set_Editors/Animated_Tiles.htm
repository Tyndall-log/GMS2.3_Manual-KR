애니메이션 타일

타일은 일반적으로 게임 룸 내에서 정적 셀로 간주되지만 GameMaker Studio 2를 사용하면 스프라이트처럼 애니메이션을 적용 할 수 있습니다. 왜 스프라이트 만 사용하지 않습니까? 스프라이트는 바운딩 박스와 렌더링 방식과 관련되어 있기 때문에 특정 처리 오버 헤드가 있지만 타일의 처리 오버 헤드가 훨씬 낮아 렌더링 속도가 더 빠릅니다. 그러나 타일 세트 애니메이션을 사용하면 길이가 2, 즉 4, 8, 16 등의 프레임 인 프레임의 애니메이션을 만드는 데 제한이 있으며 가격이 책정됩니다. 분명히 그들은 (당 타일 세트 기준)와 동일한 속도로 모든 애니메이션 있어야 타일 그리드 내에서 생성 될. 그러나 이러한 제한에도 불구하고 애니메이션 타일은 정적 환경과 배경에 생명과 색상을 추가하는 데 사용할 수있는 강력한 기능입니다.

타일 애니메이션을 만들려면 먼저 모든 하나 개의 스프라이트 이미지 내의 필요한 타일이있는 타일 세트가 있어야에서 다음과 (그러나 스프라이트 프레임으로, 모든 애니메이션이 내 이미지가 하나의 큰 스프라이트 프레임 정도) 타일 에디터 를 설정 하면 Tile Animation 버튼을 클릭하여 애니메이션 에디터를 불러옵니다 :

위의 예제 이미지는 모든 애니메이션 프레임이 포함 된 단일 이미지이지만 하나의 대규모 타일 세트를 사용하고 "체리 피킹"할 수 있기 때문에 이미지에 다른 비 애니메이션 타일도 포함될 수 있습니다. 애니메이션을 만들려는 부분. 예를 들어, 나무, 흙 및 물로 구성된 큰 가로 타일을 가질 수 있고 타일 세트 내에 물 리플을 만들기 위해 여러 타일을 애니메이션 할 수 있습니다.

타일 ​​애니메이션을 만들려면 먼저 애니메이션 추가 버튼을 클릭하여 애니메이션을 애니메이션 라이브러리에 추가해야 합니다 라이브러리에 빈 애니메이션 스트립을 추가하고 사용할 프레임 수를 선택하도록 요청합니다.

그런 다음 각 빈 애니메이션 프레임에 대한보기가 표시되며 초기 프레임이 강조 표시됩니다.

그런 다음 편집기 왼쪽의 타일 세트 이미지로 돌아 가면 타일을 애니메이션에 추가하면 프레임이 다음 타일로 넘어갑니다. 클릭 연속 타일에서 애니메이션을 채운 다음 프레임 옆의 "재생"버튼을 클릭하여 미리 볼 수 있습니다.

애니메이션이 애니메이션 라이브러리의 일부가되고 룸 편집기에 들어가면 애니메이션을 선택하여 룸 안에 배치 할 수 있습니다. 애니메이션에서 방에 배치 할 타일을 선택하면 해당 타일이 선택한 타일부터 계속 애니메이션되므로 룸 내에 별도의 프레임을 배치하여 애니메이션을 "동기화"할 수 있으며 모두 애니메이션이 적용됩니다. 이것의 유일한 경고는 애니메이션 속도가 라이브러리 내 개별 애니메이션이 아닌 전체 타일 세트에 대해 정의되므로 항상 동일하다는 것입니다.

애니메이션에 사용되는 타일에 대해 몇 가지주의 할 사항 :

  • 타일에 자체 애니메이션이 지정되어 있으면 (첫 번째 타일로 설정된 애니메이션이있는 경우) 해당 애니메이션을 사용합니다.
  • 타일에 자체 애니메이션이 없지만 하나의 다른 애니메이션에만 사용되는 경우 해당 타일이 애니메이션에있는 첫 번째 위치에서 시작하여 해당 애니메이션을 사용합니다.
  • 타일에 자체 애니메이션이없고 다른 여러 애니메이션에 사용되는 경우 전혀 애니메이션되지 않습니다. 첫 번째 타일로 애니메이션을 명시 적으로 지정해야합니다.
