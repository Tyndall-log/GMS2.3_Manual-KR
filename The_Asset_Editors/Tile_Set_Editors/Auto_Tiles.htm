자동 타일

자동 타일 기능은 하향식 또는 측면보기 프로젝트에 상관없이 레벨 구축을 훨씬 쉽게하는 데 사용할 수있는 매우 강력한 도구입니다. 기본적으로 자동 타일 라이브러리 를 만든 다음 해당 라이브러리에서 타일을 배치 할 때마다 주위의 타일과 "연결"되어 벽이나 플랫폼이 매끄럽게 만들어집니다.

그러나 자동 타일 기능을 사용하기 전에 타일 세트 스프라이트를 올바르게 설정하고 적절한 유형의 자동 타일을 선택하는 것이 매우 중요합니다. 사용 가능한 유형은 47 개의 타일 자동 타일 또는 16 개의 타일 자동 타일이며, 일반적으로 16 개의 타일은 하향식 또는 풍경에 사용되며 (전환이 더 우수하므로)보기 또는 하향식의 플랫 포머 / 측면에 사용됩니다. 벽이 여러 개의 연결을 필요로하는 곳이지만 이는 결코 어려운 규칙이 아니며 최종 프로젝트의 모양에 달려 있습니다.

아래 이미지는 자동 타일링을위한 일반적인 47 타일 및 16 타일 타일 세트를 보여줍니다.

이제 이미지를 보는 것만으로는 실제로 어떻게 어울리는 지 알려주지 않으므로 타일 ​​세트 편집기 에서 자동 편집 편집기 를 열고 추가합니다. Autotile Editor는 다음과 같습니다.

사용되는 타일 세트 스프라이트에는 필요한 모든 타일이 있으며 자동 타일에 포함 할 타일을 선택하고 선택할 수 있기 때문에 다른 타일도 가질 수 있습니다. 이제 클릭 할 수 있습니다 템플릿 영역의 47 타일 버튼을 클릭하여 새로운 47 타일 라이브러리 항목을 생성하면 필요한 타일을 선택하는 데 사용할 템플릿이 나타납니다.

템플릿에서 밝은 회색 영역은 사용중인 타일의 바깥 쪽 가장자리를 나타내고 어두운 회색은 "채우기"영역입니다. 템플릿에 타일을 추가하려면 사용 가능한 첫 번째 템플릿 셀을 클릭 한 다음 왼쪽에서 추가 할 타일을 선택하십시오.

실수 한 경우 제거하려는 자동 타일을 선택하고 타일 세트에서 "빈"타일을 선택하십시오. 모두 추가 한 후 "템플릿 표시"버튼을 클릭하여 템플릿과 타일을 확인할 수 있습니다. 아래 이미지와 같이 최종 자동 타일에서 템플릿 오버레이를 토글합니다.

16 타일 자동 타일을 만들려면 47 타일 템플릿을 사용하는 대신 16 타일 템플릿을 사용하는 것만으로 위에서 설명한 것과 정확히 동일합니다.

이제 자동 타일 라이브러리에 타일 세트에서 가져온 자동 타일이 있습니다. 타일 ​​맵 레이어를 만든 다음 라이브러리에서 자동 타일 중 하나를 선택하고 (룸 편집기 상단에서 자동 타일 브러시를 자동으로 선택 함) 룸에서 페인팅하여 룸 편집기에서 사용할 수 있습니다. 타일은 이제 "자동으로"연결되어 올바르게 순서가 지정된 타일 맵을 만듭니다.

실내 모서리 주변의 자동 타일의 동작은 열린 모서리 또는 닫힌 모서리 버튼에 따라 다릅니다. . 기본적으로 자동 타일이 방의 가장자리를 따라 배치되면 방의 외부가 비어있는 것처럼 타일이 선택됩니다 (즉, 방에 "가장자리"를 제공함). 그러나이 버튼을 클릭하면 자동 타일은 외부 영역이 타일로 채워진 것처럼 방을 타일링하므로 "가장자리"가없는 블렌드를 선택하십시오.
